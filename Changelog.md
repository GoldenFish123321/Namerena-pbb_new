# Changelog

- 修复 C++ 引擎在超大搜索区间、多前缀配置下启动崩溃的问题。

- 合并当前 `dev` 分支修复，包括 Ctrl+C 中断时刷新输出，降低结果截断风险。
- 修复 Linux CI 下 C++ 模板类型推导不一致导致的编译失败。
